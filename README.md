## 理念

- 快速执行，快速试错
- 没有完美的系统，但是有可以无限成长的系统

## 适合哪些？

- 如果你想提高自己的专注能力和事务管理能力，那么该系统适合你
- 如果你想长期在某个领域通过不断练习达成一定的成就，那么该系统适合你

## 快速记录==要做的事情==

1. 遇到想做的、要做的事情，立即按下 `打开表单` 快捷键，选择 [[🚀 Project.cform]] 表单按下回车
2. 简单的输入标题说明要做的事情
3. 然后选择一个匹配的领域
	1. 如果没有匹配的领域，请认真思考这件事情是否真的需要投入时间去做？
	2. 如果真的要做，又没有清晰的领域，请关联[[兴趣探索]]领域

### 时间戳碎碎念

1. 有一些奇怪的想法和碎碎念要记录，按下 `打开表单` 快捷键，选择 [[💡Capture.cform]] 表单按下回车
2. 输入你的想法，按下回车，内容会自动以无序列表的形式记录到当天的日记下，并在列表前添加记录时间，比如 `- 11:30 喝了一杯咖啡，好苦`

### 知识笔记

1. 直接按下 `ctrl+n` （mac 是`command+n`），会在 base 目录下创建一个文件
2. 此时你可以先选择记录内容，不用管标题和属性
3. 记录完成以后，按下 `打开表单` 快捷键，选择 [[📒 Apply basic.cform]] 来添加属性

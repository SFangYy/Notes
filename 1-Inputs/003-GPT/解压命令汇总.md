# 003-GPT
文件格式解析
1. **.tar**：将多个文件打包成一个单一的 `.tar` 文件（归档文件）。
2. **.xxx**：使用 `不同的算法` 对这个 `.tar` 文件进行压缩。

通用命令

### tar.gz 
- 压缩 
	- `-c`：**创建**新的归档文件。
	- `-z`：使用 **gzip** 对归档文件进行压缩。
	- `-f`：指定归档**文件**名。这个参数后面必须紧跟你自己设定的文件名。
	- `-v` 参数。这会在终端中列出所有正在被添加到压缩包中的文件
- 解压
	- `-x`：**提取**（解包）文件。
	- `-z`：使用 **gzip** 进行解压缩。
	- `-f`：指定**文件**名。这个参数后面必须紧跟文件名。
```
tar -xzf file.tar.gz
tar -czf archive_name.tar.gz path_to_file_or_directory
```

### tar.bz2
`bzip2` 通常能提供比 `gzip` 更高的压缩率（文件更小），但压缩和解压速度会慢一些。
- 压缩
- `-c`：**创建**新的归档文件。
- `-j`：使用 **bzip2** 进行压缩。
- `-f`：指定归档**文件**名。
- 解压
	- `-x`：**提取**（解包）文件。
	- `-j`：使用 **bzip2** 进行解压缩。
	- `-f`：指定**文件**名。

```
tar -xjf archive.tar.bz2 -C /path/to/target/directory


```

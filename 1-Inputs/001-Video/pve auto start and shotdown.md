# 001-Video

use `crobtab` tools to start config service
```
crobtab -e
```
then choose a editor and then add two line as follows

```
00 08 * * 1-7 /usr/sbin/qm start 101 > /dev/null 2>&1
00 01 * * 1-7 /usr/sbin/qm shutdown 101 > /dev/null 2>&1
```

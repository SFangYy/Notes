# QuickNotes
**2. 常用快捷键：**

- 目录导航：

- `j`：向下移动光标
- `k`：向上移动光标
- `h`：返回上一级目录
- `l`：进入选中目录

- 处理文件：

- `d`：删除选中文件
- `x`：剪切选中文件
- `y`：复制选中文件
- `p`：粘贴文件
- `r`：重命名文件

- 其他：

- `.`：显示隐藏文件
- `q`：退出
- `~`：打开帮助
- `Z`：使用 `[fzf]

跳转到目录或显示文件（需要安装 fzf ）

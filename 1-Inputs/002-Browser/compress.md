# Resources

## tar
compress file or dictory
`tar -czvf example.tar.gz example/`

extracting  file
`tar -zxvf example.tar.gz`

## zip
compress dictory
```
zip -er xxx.zip dictionary
zip -er backup.zip backup

```
extracting  file
`unzip xxx.zip`

# browser

## 为什么要聊天
- 信息`交换`
	- 提供信息给对方，让对方传递信息给我们
- 不知道聊什么的原因是因为信息交换的太少了
- 交换是两个人互相进行

## 建立信息交换的聊天机制
- 尽量找对方感兴趣的关键词
	- 不罗嗦，不打扰
- 抛出问题，引导对方回答
	- 我想唱，你之前说的xxx也不错，你感觉呢
- 关键词，围绕对方
	- 以我对你的认识，你是那种会骑行到西藏的人
- 反映力练习
	- 周围的任何东西，通过她,在10秒中内联想出一个故事，每天5组

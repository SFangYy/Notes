
配置 KVM 桥接网络（Bridge Networking）可以让虚拟机像一台真实的物理机一样连接到你的路由器，获得与宿主机同网段的独立 IP。

**核心逻辑：** 你需要先在**宿主机（Host）的操作系统**层面创建一个“网桥（Bridge）”设备（比如 `br0`），把物理网卡“插”进去。然后在 KVM 设置里让虚拟机使用这个 `br0`。

由于不同的 Linux 发行版管理网络的方式不同，请根据你的宿主机系统选择 **Step**
```
nmcli con add type bridge ifname br0 con-name br0
nmcli con add type bridge-slave ifname enp0s31f6 master br0
# 先把原本的连接停掉（如果你是 SSH 远程，这一步可能会让你断线！）
nmcli con down "Wired connection 1"

# 启动网桥连接
nmcli con up br0


```

### Step 2: 修改虚拟机配置

一旦宿主机有了 `br0`，就可以把虚拟机挂上去了。

#### 方法一：使用 `virt-manager` (图形界面)

1. 打开虚拟机详情 -> 灯泡图标 (Hardware)。
    
2. 点击左侧的 **"NIC" (网卡)**。
    
3. **Network source (网络源):** 选择 **"Bridge device: br0"** (桥接设备 br0)。
    
    - _如果没有 br0 选项，选 "Specify shared device name" 手动输入 `br0`。_
        
4. **Device model (设备型号):** 推荐选 **virtio** (性能最好)。
    
5. 点击 Apply，重启虚拟机。

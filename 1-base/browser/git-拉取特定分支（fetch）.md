## 拉取特定远程分支 
还是要先添加upstream之后，可以通过如下步骤，实现拉去特定分支
```
✦ 已成功将原始仓库的 kunminghu-v3 分支拉取到本地！

  现在你可以：
   - 查看该分支：git checkout upstream/kunminghu-v3
   - 创建本地分支：git checkout -b kunminghu-v3 upstream/kunminghu-v3
   - 合并到当前分支：git merge upstream/kunminghu-v3

然后可以通过git push origin 推送到自己的仓库
```

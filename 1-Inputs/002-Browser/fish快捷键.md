# 002-Browser
```
C-a 光标回到行首, C-e 光标回到行尾
C-f 光标向前移动一个字符，C-b 光标向后移动一个字符
Alt-f 光标向前移动一个单词, Alt-b 光标向后移动一个单词
C-k 清掉光标后面的部分, C-d 删掉光标后的一个字符 或 退出
C-l 相当于 clear 命令，清屏

转载请注明原文链接: https://writings.sh/post/commandline-tools
```
